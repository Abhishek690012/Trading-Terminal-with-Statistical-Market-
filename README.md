# Stock Trading Simulation System

A full-stack application that simulates a stock trading environment with order matching, portfolio tracking, and dynamic price updates.

---

## Overview

This project allows users to:

* Place buy and sell orders
* Track their portfolio
* View changing stock prices
* Experience a basic order matching system

The focus is on implementing core backend logic and system design principles.
